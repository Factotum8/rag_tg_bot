# Как это работает на практике
**Запускаешь контейнер — он сам каждые N минут/часов обновляет индекс.**    
При изменении JSON:
* изменится hash → скрипт удалит старые чанки этого документа и добавит новые.
* При удалении документа из JSON (если DELETE_STALE=true) — удалит чанки.
